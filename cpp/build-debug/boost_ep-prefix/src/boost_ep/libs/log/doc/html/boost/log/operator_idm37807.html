Function template operator<<

boost::log::operator<<

Synopsis

// In header: <boost/log/utility/manipulators/range.hpp>


template<typename StreamT, typename RangeT, typename DelimiterT> 
  unspecified operator<<(StreamT & strm, 
                         range_manipulator< RangeT, DelimiterT > const & manip);

Description

Stream output operator for range_manipulator. Outputs every element of the range, separated with a delimiter, if one was specified on manipulator construction.
